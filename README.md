# Scientific Calculator
Overview
- A web-based scientific calculator built using HTML, CSS, and JavaScript. 
- This project enables users to perform basic arithmetic and advanced mathematical operations with a responsive and intuitive interface, optimized for both desktop and mobile devices.
## Features
- Basic arithmetic operations (addition, subtraction, multiplication, division, percentage).
- Advanced mathematical functions (trigonometric, logarithmic, exponential, square root).
- Responsive design for seamless use across devices.
- Interactive and user-friendly interface with real-time result display.
- Robust error handling for invalid inputs.
## Technologies Used
- HTML: Structures the calculator layout.
- CSS: Provides modern, responsive styling.
- JavaScript: Powers the calculation logic and user interactions.
